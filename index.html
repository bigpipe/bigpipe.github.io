Temper Temlates: Temper is a small module that compiles your templates for server-side usage and client-side usage through one single interface. This makes it easy to create isomorphic JavaScript applications, which is awesome. The following template engines are supported: react-jsx, automatically discovered by using the .jsx extension. jade, automatically discovered by using the .jade extension. ejs, automatically discovered by using the .ejs extension. hogan.js, automatically discovered by using the .mustache extension. mustache, automatically discovered by using the .mustache extension. handlebars, automatically discovered by using the .mustache extension. html, automatically discovered by using the .html extension.
